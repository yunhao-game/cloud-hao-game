# Cloud Hao Game - 云和皓的第一个游戏

一款基于 React Native + Expo 开发的自走棋策略游戏。

## 游戏简介

玩家通过商店购买英雄棋子，将棋子拖拽到棋盘上布阵，战斗自动进行。击败敌人后推进关卡，挑战更强对手。

## 技术栈

- React Native + Expo
- TypeScript
- Android SDK

## 游戏玩法

1. 商店购买英雄棋子
2. 拖拽棋子到棋盘布阵（8x8棋盘，下半区是我方）
3. 战斗自动进行，棋子自动寻找敌人并攻击
4. 击败敌人后推进关卡，挑战更强对手
5. 击败第4张地图Boss后获得胜利
